### 1. Clone the repo

```bash
git clone https://github.com/DoNguyenAnhTuan/StandeeDashboard_Carbon.git
cd StandeeDashboard_Carbon

2. Install dependencies

npm install

3. Run Python dependencies (optional)
Ensure you have Python + the following libraries:

pip install -r requirements.txt
# Or manually:
pip install pandas requests xgboost scikit-learn

4. Start the server

npm run dev

The scripts will:
  Auto fetch site data
  Run forecast
  Launch the server at: http://localhost:3000
